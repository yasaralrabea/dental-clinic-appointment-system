- Dental Clinic Appointment Booking System

A web-based appointment booking system for dental clinics, built with Laravel and MySQL.  
The system allows patients to book appointments, communicate with the clinic, and manage their schedules, while administrators have full control through a powerful dashboard.

---

 🚀 Features
- 🔐 User Authentication** (Login, Register, Password Reset)
- 📅 Appointment Booking System**
- 💬 Messaging System** between patients and clinic
- 📊 Admin Dashboard** with full control
- 👩‍⚕️ Manage doctors, patients, and appointments
- 📧 Email notifications
- 🗂️ Database migrations & seeders
- 🖼️ Blade templating with clean UI



 🛠️ Tech Stack
- [Laravel](https://laravel.com/) – PHP Framework
- [MySQL](https://www.mysql.com/) – Database
- [Composer](https://getcomposer.org/) – Dependency Manager
- [Artisan CLI](https://laravel.com/docs/artisan) – Laravel Command Line Tool
- [Bootstrap / Tailwind] (choose what you used) – Frontend styling
